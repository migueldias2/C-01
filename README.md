# C-01

Collection of C practice exercises.

## Download

```bash
git clone https://github.com/mcarepa-42/C-01.git
cd C-01
```

You can also download the ZIP from GitHub.

## Requirements

Install a C compiler such as `cc`, `gcc`, or `clang`.

## Build and run

Go into one exercise folder and compile its files:

```bash
cd ex00
cc -Wall -Wextra -Werror *.c -o test
./test
```

When a function has no test program, create a small `main.c` in that folder and compile it together with the exercise file.
